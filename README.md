# <img src="/BV6Tools/Assets/bv6tools-icon-32.png"> BV6Tools
A simple WPF for easy use with GL, ST, OST.

## Features
- Search AppId by Name for directly add to Applist.
- Automatic Cleanup after steam exit.
- Ticket Dumping (AppOwnership and ETicket).

## Requirements
- Windows 10, Version 1809 (Build 17763) 64-bit

## Download
Grab the latest release from the [Releases](../../releases) page.

It's fully portable, no installer required.
> For usage details, check the in-app **Guide** page.
